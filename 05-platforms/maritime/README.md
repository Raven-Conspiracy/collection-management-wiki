## 10. Maritime Collection Platforms

### 10.1 MQ-8C Fire Scout (Ship-Based VTOL UAV)

| Field | Details |
|---|---|
| **Designation** | MQ-8C Fire Scout (larger Bell 407 derivative); MQ-8B (smaller Schweitzer-derived — legacy) |
| **Operator** | USN (deployed on LCS, DDG-51 Flight III, and amphibious ships; also land-deployed for counter-IED) |
| **Type** | Group 4 Ship-Based Autonomous VTOL UAV |
| **Collection Capability** | Multi-mode maritime surface search radar; EO/IR targeting/surveillance sensor; AIS; APKWS-compatible (weapons option); laser designator; provides ISRT (ISR and Targeting) |
| **Sensor Names** | Multi-mode maritime surface search radar; EO/IR sensor pod |
| **Performance Envelope** | Length: 41.4 ft (12.6 m) MQ-8C; Rotor diameter: 35 ft; Max MTOW: 6,000 lb; Payload: 700+ lb; Endurance: 12 hours (300 lb payload); Max speed: 135 kts; Cruise: 115 kts; Ceiling: 16,000 ft; Range: 150 nm |
| **Datalinks / Comms** | CDL; SATCOM relay; ship-compatible GCS; identical ship-based and portable ground hardware |
| **PED Architecture** | Ship's combat system; DCGS-N afloat |
| **Tasking Authority** | Ship CO; CFMF; Theater N2 |
| **Primary Mission Set** | Ship-based organic maritime ISR; over-the-horizon targeting; surface search and surveillance; manned-unmanned teaming with manned helicopters (MH-60R/S); Expeditionary Advanced Base Operations (EABO) |
| **Limitations** | Limited to sea-state conditions for launch/recovery; weather limitations; relatively slow speed |
| **Program Status** | Active; deployed on LCS and DDG; Fire Scout MQ-8B legacy fleet transitioning |
| **Public Sources** | [Navy Fact File MQ-8C](https://www.navy.mil/Resources/Fact-Files/Display-FactFiles/Article/2159302/mq-8c-fire-scout/) | [Northrop Grumman Fire Scout](https://www.northropgrumman.com/what-we-do/aircraft/fire-scout) |

### 10.2 T-AGOS / SURTASS (Undersea Surveillance Ships)

| Field | Details |
|---|---|
| **Designation** | T-AGOS (Tactical Auxiliary General Ocean Surveillance); current class: Impeccable (T-AGOS-23); TAGOS-25 class (new construction, Austal USA) |
| **Operator** | Military Sealift Command (MSC) civilian-crewed; USN operational control; supports IUSS (Integrated Undersea Surveillance System) |
| **Type** | Ocean Surveillance Ship — Maritime ACINT/MASINT collection |
| **Collection Capability** | **SURTASS** (Surveillance Towed Array Sensor System) — passive broadband and low-frequency linear hydrophone array towed behind ship; CLFA (Compact Low Frequency Active sonar) — active sonar component; data transmitted via SATCOM to IUSS shore processing centers and ASW commanders in near-real-time; mobile augmentation to fixed SOSUS bottom arrays |
| **Performance Envelope** | Towing speed: ~3–6 knots (low speed required for passive acoustics); SURTASS array length: ~1.8 km; CLFA frequency: ~100–500 Hz active; Range of detection: classified (hundreds to thousands of km for quiet submarines via SOFAR channel) |
| **Datalinks / Comms** | SATCOM data link to shore-based processing centers (Naval Ocean Processing Facilities — NOPFs); HF communications |
| **PED Architecture** | IUSS (Integrated Undersea Surveillance System); Naval Ocean Processing Facility (NOPF) Moffett Field CA and Norfolk VA; NAVSEA PMS-485 |
| **Tasking Authority** | Fleet commanders (CINCLANTFLT/CINCPACFLT); ASW commander; theater N2 |
| **Primary Mission Set** | Passive acoustic detection and tracking of submarines; mobile SOSUS augmentation in areas lacking fixed arrays; cues prosecution forces (P-8A, SSN, surface ships) |
| **Limitations** | Must operate at very low speeds (vulnerable to sea states and adversary targeting); long tow geometry limits maneuverability; passive detection range degraded in thermocline/acoustic noise environments; Chinese maritime harassment documented during SURTASS operations |
| **Public Sources** | [Austal TAGOS-25](https://www.austalusa.com/ships/surface-ships/tagos-25-navy-ocean-surveillance-ship) | [USNI SOSUS history](https://www.usni.org/magazines/naval-history-magazine/2021/february/66-years-undersea-surveillance) | [DOT&E SURTASS/CLFA](https://www.dote.osd.mil/Portals/97/pub/reports/FY2011/navy/2011surtass.pdf) |

### 10.3 Sea Hunter / MUSV (Medium Unmanned Surface Vehicle)

| Field | Details |
|---|---|
| **Designation** | Sea Hunter I and Sea Hunter II (MUSV — Medium Displacement Unmanned Surface Vehicle) |
| **Operator** | ONR (Office of Naval Research); transitioning to NAVSEA/PEO USC |
| **Type** | Autonomous Unmanned Surface Vessel — Anti-Submarine Warfare / ISR |
| **Collection Capability** | Anti-submarine warfare (ASW) continuous trail mission; acoustic sensors for submarine detection and tracking; designed to trail diesel-electric submarines for weeks; payload bay for future sensors/weapons |
| **Performance Envelope** | Length: 132 ft; Displacement: 145 tons; Max speed: 27 kts; Range: 10,000 nm; Fully autonomous navigation (COLREGs-compliant); cost: ~$23M prototype vs $700K/day destroyer |
| **Datalinks / Comms** | SATCOM; sparse remote supervisory control from shore |
| **PED Architecture** | ONR/NAVSEA operational control; data feeds to ASW processing ashore |
| **Tasking Authority** | Fleet ASW commander; NAVSEA PEO USC |
| **Primary Mission Set** | Persistent autonomous submarine trailing; cost-effective alternative to crewed ASW ships; forward presence in contested waters |
| **Limitations** | Still developmental/transitional; autonomous navigation in contested environments; limited payload diversity currently; cyber vulnerability potential |
| **Program Status** | Sea Hunter I and II prototypes operational testing; MUSV program of record in development |
| **Public Sources** | [Naval Technology Sea Hunter](https://www.naval-technology.com/features/sea-hunter-inside-us-navys-autonomous-submarine-tracking-vessel/) | [Leidos MDUSV program](https://www.maritime.dot.gov/sites/marad.dot.gov/files/docs/about-us/foia/11706/leidos-inc-sea-hunter-and-maritime-autonomous-behaviors.pdf) |

### 10.4 AN/SPY-6 Aegis Radar (DDG-51 Flight III) — Collection-Relevant Sensor

| Field | Details |
|---|---|
| **Designation** | AN/SPY-6(V)1 (Flight III DDG); also AN/SPY-6(V)3 (LCS/smaller ships); family includes AMDR variant |
| **Operator** | USN (DDG-51 Flight III Arleigh Burke class; CG-47 Ticonderoga class with AN/SPY-1D legacy) |
| **Type** | Active Electronically Scanned Array (AESA) Multi-Function Radar — ISR-relevant |
| **Collection Capability** | Volume search and tracking of air/ballistic/hypersonic targets; maritime surface search; discrimination of ballistic missile warheads from debris (MD cueing); passive/active ELINT-adjacent (electronic intelligence from radar returns); target track files for indications & warnings; data fusion with DCGS-N and Aegis combat system |
| **Sensor Names** | AN/SPY-6(V)1 S-band AESA (volume search); X-band precision tracking radar; AN/SPQ-9B surface search (legacy ships) |
| **Collection Manager Note** | Aegis ships are not dedicated intelligence collectors, but their sensor suites generate significant intelligence products: electronic order of battle data, air tracks, BMD discrimination data shared via Link-16 and DCGS-N with theater intelligence architecture |
| **Public Sources** | [Wikipedia AN/SPY-6](https://en.wikipedia.org/wiki/AN/SPY-6) | [NAVSEA SPY-6 article](https://www.navsea.navy.mil/Media/News/Article-View/Article/4244137/anspy-6v1-radar-eyes-of-the-fleet/) |

### 10.5 SSN / SSGN Intelligence Collection Role

| Field | Details |
|---|---|
| **Designation** | SSN (Fast Attack Submarine): Virginia-class (SSN-774+); Los Angeles-class (SSN-688); SSGN: Ohio-class conversion (SSGN-726 to 729) |
| **Operator** | USN (SUBLANT/SUBPAC) |
| **Type** | Nuclear-Powered Submarine — Multi-Mission Intelligence Collector |
| **Collection Capability (UNCLASSIFIED)** | **ACINT (Acoustic Intelligence)**: passive sonar arrays (bow sphere, wide-aperture array, towed array TB-16/29/33) for acoustic signature collection on adversary submarines/surface ships; contributes to acoustic intelligence database; **SIGINT/ELINT**: AN/BLQ-10 ESM mast — intercepts radar/comms while at periscope depth; **Periscope/Photonics Mast**: optical/IR imagery collection of shore facilities, surface ships, aircraft; **HUMINT enabling**: SEAL delivery, special operations insertion/extraction; contributes to seabed mapping and bathymetric intelligence |
| **Sensor Names** | AN/BQQ-10 (Virginia) sonar; TB-29 towed array; photonics mast EO/IR; AN/BLQ-10 ESM |
| **Collection Manager Note** | SSNs are among the most capable and covert intelligence collectors. Collection is validated against specific theater and national requirements. ACINT contributions to submarine acoustic intelligence databases are national-level products. |
| **Limitations** | Classified collection architecture; must surface/periscope for some collection; geographically limited by access and rules of engagement |
| **Public Sources** | [USNI SOSUS/submarine history](https://www.usni.org/magazines/naval-history-magazine/2021/february/66-years-undersea-surveillance) | [ACINT definition](https://www.emsopedia.org/entries/acoustic-intelligence-acint/) |
