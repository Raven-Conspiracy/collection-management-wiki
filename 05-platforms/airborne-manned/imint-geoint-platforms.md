## 4. IMINT / GEOINT — Airborne Manned Platforms

### 4.1 U-2S Dragon Lady

| Field | Details |
|---|---|
| **Designation** | U-2S (modernized); U-2ST (two-seat trainer) |
| **Operator** | USAF (9th Reconnaissance Wing, Beale AFB, CA; Det 1 Osan AB Korea; deployed worldwide) |
| **Type** | Manned HALE Strategic Reconnaissance Aircraft |
| **Collection Capability** | **SYERS-2C** (Senior Year Electro-Optical Reconnaissance System — 10-band multispectral/VNIR/SWIR/LWIR, completed fleet upgrade Feb 2020); **ASIP** (Airborne Signals Intelligence Payload) SIGINT; **SENIOR RUBY / SENIOR SPAN** data links; SAR capable; MASINT sensors; optical cameras; **MARS** (Multi-spectral Advanced Radar System); Senior Glass EO system; HMMWV-compatible ground exploitation |
| **Sensor Names** | SYERS-2C multispectral EO; ASIP SIGINT; Advanced Synthetic Aperture Radar System (ASARS-2A) |
| **Performance Envelope** | Ceiling: 70,000+ ft MSL (near-space); Endurance: ~10–12 hours loiter; Range: 7,000 nm; Cruise speed: ~410 kts; Payload: 3,500 lb (multiple simultaneous bays); Single GE F118-101 turbofan engine |
| **Datalinks / Comms** | SENIOR SPAN (Ka-band SATCOM near-real-time downlink); SENIOR RUBY LOS relay; CDL; Link-16; HF/VHF voice |
| **PED Architecture** | AF DCGS (primary); NSA for SIGINT; NGA for IMINT products; dedicated U-2 mobile ground stations (MOGS/TOGS) |
| **Tasking Authority** | JFACC/CFACC via ATO; national-level tasking (NRO/DIA integration); CCMD J2 |
| **Primary Mission Set** | Strategic all-weather IMINT (multispectral, SAR); national-level SIGINT; MASINT; indications & warnings; treaty verification; deep-penetration standoff collection in denied areas |
| **Limitations** | Difficult to fly in contested airspace given non-stealthy profile; extreme altitude requires specialized PPE (pressure suit); single-pilot workload; weather affects optical collection; pilot handling challenges (bicycle landing gear + pogo gear) |
| **Program Status** | Active; USAF proposed retirement multiple times but Congress has sustained; SYERS-2C upgrade fleet-wide complete 2020; continues to fly contested ISR missions |
| **Public Sources** | [AF Fact Sheet](https://www.af.mil/About-Us/Fact-Sheets/Display/Article/104560/u-2stu-2s/) | [FAS U-2](https://irp.fas.org/program/collect/u-2.htm) | [Lockheed SYERS-2C](https://news.lockheedmartin.com/2020-02-18-U-S-Air-Force-Lockheed-Martin-Collins-Aerospace-Complete-U-2-SYERS-2C-Upgrades-Enhancing-the-Dragon-Ladys-Imaging-Capability) |
