## 5. SIGINT — Airborne Manned Platforms

### 5.1 RC-135V/W Rivet Joint

| Field | Details |
|---|---|
| **Designation** | RC-135V (mission 1 config); RC-135W (mission 2 config) |
| **Operator** | USAF (55th Wing, Offutt AFB, NE; detachments worldwide); RAF co-operated RC-135W Airseeker (3 aircraft, ~2035) |
| **Type** | Manned Strategic/Theater SIGINT Aircraft |
| **Collection Capability** | Full-spectrum SIGINT: COMINT (communications interception, geoloaction, demodulation); ELINT (radar/emitter identification and geolocation); rapid search, detection, identification across electromagnetic spectrum; on-board analysis and dissemination; AEELS (Automated ELINT Emitter Locating System) in cheek pods |
| **Sensor Names** | AEELS cheek pod arrays; high- and low-band COMINT antennas; ELINT receivers; classified sensor suite managed by L3Harris/L3 Technologies |
| **Performance Envelope** | Endurance: 12+ hours (with aerial refueling: unlimited); Altitude: 35,000–40,000 ft; Range: 3,900+ nm; Powered by 4× CFM International F108-CF-201 turbofans (21,600 lb ea); MTOW: 297,000 lb; Crew: 30+ (pilots, EW officers, intel operators, in-flight maintenance) |
| **Datalinks / Comms** | HF/VHF/UHF voice; SATCOM; dedicated intelligence data links for product dissemination; JWICS-capable |
| **PED Architecture** | NSA/CSS (national SIGINT); AF DCGS for theater SIGINT products; 55 WG intelligence operations; direct reporting to national and theater customers |
| **Tasking Authority** | National: NSA/JS J2 directives; Theater: JFACC via ATO, CCMD J2 |
| **Primary Mission Set** | Theater and strategic ELINT/COMINT collection; emitter geolocation; near-real-time intelligence to tactical/operational commanders and national consumers; RAF partnership for European/NATO operations |
| **Limitations** | Large, non-stealthy aircraft — standoff operations required in contested airspace; valuable target for adversary air defenses; large crew requirement; some signals require proximity that may not be achievable |
| **Program Status** | Active; fleet modernized with CFM-56 engines and "glass cockpit" digital instrumentation; extensive combat experience Desert Storm through present |
| **Public Sources** | [AF Fact Sheet](https://www.af.mil/About-Us/Fact-Sheets/Display/Article/104608/rc-135vw-rivet-joint/) | [FAS Rivet Joint](https://irp.fas.org/program/collect/rivet_joint.htm) | [Air & Space Forces](https://www.airandspaceforces.com/weapons/rc-135v-w/) |

### 5.2 RC-135U Combat Sent

| Field | Details |
|---|---|
| **Designation** | RC-135U Combat Sent (2 aircraft in fleet) |
| **Operator** | USAF (55th Wing, Offutt AFB) |
| **Type** | Manned Strategic ELINT / TECHINT Aircraft |
| **Collection Capability** | Strategic technical intelligence (TECHINT) collection on adversary radar/emitter systems; AEELS (Automated ELINT Emitter Locating System) in chin and wingtip arrays; individual tailored sensor suites per airframe; data for RWR, jammer, decoy, anti-radiation missile programming |
| **Sensor Names** | AEELS arrays; specialized emitter analysis suite; classified TECHINT sensors |
| **Performance Envelope** | Same airframe as RC-135V/W; aerial refueling enables unlimited range; Crew: similar to Rivet Joint with ELINT specialists |
| **Datalinks / Comms** | SATCOM; HF/VHF/UHF data links; direct connectivity to TECHINT customers |
| **PED Architecture** | NSA/CSS; DIA MSIC (Missile and Space Intelligence Center); national-level TECHINT consumers; 55 WG intelligence |
| **Tasking Authority** | National: NSA/JS J2; SECDEF directives; JCS-directed missions |
| **Primary Mission Set** | Foreign radar system characterization; RWR/jammer threat database development; strategic ELINT for NCA and combatant commanders |
| **Limitations** | Only 2 aircraft (extremely high-demand/low-density asset); standoff required; missions are national-level priority with significant scheduling competition |
| **Program Status** | Active but very small fleet; extremely high priority for national collection |
| **Public Sources** | [AF Fact Sheet Combat Sent](https://www.af.mil/About-Us/Fact-Sheets/Display/Article/104495/rc-135u-combat-sent/) | [Air & Space Forces](https://www.airandspaceforces.com/weapons/rc-135u/) |

### 5.3 RC-12X Guardrail Common Sensor (Army)

| Field | Details |
|---|---|
| **Designation** | RC-12X/X+ Guardrail (most current; also RC-12K/N/P/Q legacy) |
| **Operator** | USA (Military Intelligence Brigades — typically 8–12 aircraft per battalion in sets of 2–3) |
| **Type** | Manned Corps/Theater-Level Airborne SIGINT Aircraft |
| **Collection Capability** | **GRCS (Guardrail Common Sensor):** Integrated COMINT (Improved Guardrail V — IGR V); ELINT (Advanced QUICKLOOK — AQL); COMINT High Accuracy Airborne Location System (CHAALS) for precision geolocation; near-real-time SIGINT and targeting; RC-12Q adds SATCOM dorsal radome for BLOS connectivity |
| **Sensor Names** | GRCS System (IGR V + CHAALS + AQL); RC-12X has GRCS System latest generation |
| **Performance Envelope** | Based on Beechcraft Super King Air C-12 platform; Ceiling: ~31,000 ft; Endurance: ~6–8 hours; Typical echelon: Corps level |
| **Datalinks / Comms** | SATCOM (RC-12Q+); TROJAN SPIRIT ground relay; LOS data link to ground stations |
| **PED Architecture** | Army MI Brigade ground exploitation stations; DCGS-A; NSA/CSS for national-level COMINT/ELINT |
| **Tasking Authority** | ASCC G2; Corps G2; MI Brigade commander |
| **Primary Mission Set** | Corps-level airborne SIGINT; enemy comms interception, ELINT, precision emitter geolocation; targeting data to fires and maneuver commanders |
| **Limitations** | Aging airframe (GRCS useful life extended to 2034); turboprop platform limits speed and altitude vs adversary threats; being replaced by HADES program |
| **Program Status** | Active; economic useful life extended; transition to HADES pending |
| **Public Sources** | [FAS Guardrail](https://irp.fas.org/program/collect/guardrail.htm) | [Army Article](https://www.army.mil/article/23685/guardrail_improvements_enhance_intelligence_support_and_deployability) | [Government Procurement RC-12X](https://www.governmentprocurement.com/news/all-about-the-rc-12x-guardrail) |

### 5.4 EO-5C ARL-M (Airborne Reconnaissance Low — Multifunction)

| Field | Details |
|---|---|
| **Designation** | EO-5C ARL-M (Airborne Reconnaissance Low — Multifunction); De Havilland DHC-7-based |
| **Operator** | USA (INSCOM military intelligence units; deployed worldwide) |
| **Type** | Manned Multi-Intelligence Low-to-Medium Altitude ISR Aircraft |
| **Collection Capability** | COMINT collection and geolocation; GMTI/SAR radar; EO/IR FMV (full motion video); IMINT; onboard analysis and real-time dissemination from onboard open-architecture workstations; described as "most densely-packed sensor plane in the world" |
| **Sensor Names** | COMINT suite (classified); GMTI/SAR radar; EO/IR FMV sensor pod |
| **Performance Envelope** | 4-engine turboprop DHC-7 airframe; Endurance: limited by fuel and crew duty; Low-to-medium altitude operations (under radar threats); designed for low-profile/low-observable profile |
| **Datalinks / Comms** | Satellite comms; real-time downlink to ground stations; JWICS connectivity |
| **PED Architecture** | INSCOM ground stations; DCGS-A integration; onboard real-time analysis |
| **Tasking Authority** | INSCOM commander; theater J2/G2; can support CCMD requirements |
| **Primary Mission Set** | Multi-INT collection for counternarcotics, counterterrorism, counterinsurgency; border surveillance; ground force movement tracking; littoral surveillance; low-profile deep area collection |
| **Limitations** | DHC-7 platform aging; limited speed and altitude; not survivable in contested airspace; transition to HADES program underway |
| **Program Status** | Legacy system; actively being replaced by HADES; reduced fleet |

### 5.5 HADES — High Accuracy Detection and Exploitation System

| Field | Details |
|---|---|
| **Designation** | HADES (High Accuracy Detection and Exploitation System); based on Bombardier Global 6500 business jet |
| **Operator** | USA (Army ISR Task Force → Army Intel and Security Command acquisition; up to 11 aircraft planned) |
| **Type** | Manned High-Altitude ISR Aircraft (business jet derivative) |
| **Collection Capability** | ELINT and COMINT sensors (L3Harris ARES/HADES sensor suite); Multi-Domain Sensing System (MDSS) architecture; AI/ML-enabled onboard processing by Sierra Nevada Corporation (SNC); designed for deep sensing in multi-domain operations; sensors engineered for contested/denied environments |
| **Sensor Names** | L3Harris ARES/HADES ELINT/COMINT suite; SNC AI/ML-enabled mission processing exploitation and dissemination |
| **Performance Envelope** | Based on Bombardier Global 6500: Ceiling: ~51,000 ft; Range: ~6,600 nm; Endurance: 14+ hours (18+ hours with tankering); Speed: Mach 0.88 cruise (~480 kts); Payload capacity significantly greater than legacy turboprop ISR aircraft |
| **Datalinks / Comms** | SATCOM BLOS; assured/denied/degraded comms survivable architecture |
| **PED Architecture** | DCGS-A; Army MIOC; onboard PED capability (AI/ML fusion) |
| **Tasking Authority** | ASCC/ARFOR; CCMD theater support; designed for deep sensing vs peer/near-peer forces |
| **Primary Mission Set** | Deep sensing in A2/AD environments; SIGINT collection against peer forces; replace RC-12 Guardrail/EO-5C legacy fleet; strategic aerial ISR for Army and joint force |
| **Limitations** | Still in development/early fielding; sensor integration maturing; program budget/quantity under review; high cost per aircraft vs legacy fleet |
| **Program Status** | First aircraft delivered to Army Nov 2024; SNC as Lead Systems Integrator; up to 11 aircraft planned; replacing RC-12 and EO-5C fleet |
| **Public Sources** | [Bombardier Defense Delivery](https://bombardier.com/en/media/news/bombardier-defense-delivers-first-global-6500-aircraft-us-armys-hades-program) | [Army.mil HADES](https://www.army.mil/article/265353/hades_modernizes_aerial_military_intelligence) | [Defense Scoop HADES](https://defensescoop.com/2024/10/15/army-hades-business-jet-isr-program-pre-prototypes-deployments/) | [L3Harris ARES datasheet](https://www.l3harris.com/sites/default/files/2022-03/ims-isr-ARES-HADES-data-sheet.pdf) |

### 5.6 P-8A Poseidon (Maritime ISR / ASW)

| Field | Details |
|---|---|
| **Designation** | P-8A Poseidon |
| **Operator** | USN (Patrol Squadrons — VP; ~120 aircraft; bases: NAS Jacksonville FL, NAS Whidbey Island WA, MCAS Kaneohe Bay HI, NAF Atsugi Japan; allied nations) |
| **Type** | Manned Maritime Patrol and Reconnaissance Aircraft |
| **Collection Capability** | **AN/APY-10 SAR/ISAR** multi-mode radar (SAR imagery, ISAR vessel classification, inverse SAR); **EO/IR sensor** (forward-looking, targeting); **Acoustic systems**: sonobuoy launchers (multiple types — DIFAR, DICASS, VLAD, BT, active) + advanced signal processing; **AN/APS-137D(V)5** inverse SAR and ISAR radar (in some configs); **ESM** suite; **MAD-XR** magnetic anomaly detector (some international operators; USN deleted MAD requirement but integration possible); AIS; HF/UHF/SATCOM comms relay |
| **Sensor Names** | AN/APY-10 AESA radar; AN/APS-137 ISAR; EO/IR targeting system; acoustic sonobuoy system; ESM receivers |
| **Performance Envelope** | Based on Boeing 737-800ERX; Ceiling: 41,000 ft; Max speed: 490 kts; Cruise: ~440 kts; Range: 1,200 nm loiter at 3 hrs from base; Crew: 9 (2 pilots, 1 flight engineer, 3–5 tactical operators) |
| **Armament** | AGM-84 Harpoon (anti-surface); Mk-54 torpedoes (ASW); mines; depth charges; internal weapons bay |
| **Datalinks / Comms** | Link-16; CDL; SATCOM; ROVERS; JWICS-capable ground system integration |

| **Program Status** | Active primary USN maritime patrol aircraft; ~120 USN aircraft; P-3C fully retired |
