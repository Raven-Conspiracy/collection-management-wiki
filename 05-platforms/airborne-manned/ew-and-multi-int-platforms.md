## 9. Electronic Warfare / Multi-INT — Airborne Platforms

### 9.1 EA-18G Growler (SIGINT-Relevant EW)

| Field | Details |
|---|---|
| **Designation** | EA-18G Growler |
| **Operator** | USN (VAQ squadrons embarked on CVN and at NAS Whidbey Island WA); RAAF (12 aircraft) |
| **Type** | Carrier-Based Electronic Attack Aircraft (EW + SIGINT support) |
| **Collection Capability** | **AN/ALQ-218(V)2** wideband receiver (wingtips) — passive SIGINT collection across broad frequency range; identifies, geolocates, and characterizes emitters for immediate and archival use; **AN/APG-79 AESA** radar (provides targeting, SAR-like imaging); **ALQ-99** high-/low-band jamming pods (also function as ESM collectors during passive modes); Block II **ICAP-D** improvements: advanced EW suite; AN/ALQ-227 CDSM; HARM-enabled SEAD |
| **Sensor Names** | AN/ALQ-218(V)2 wideband receiver; AN/APG-79 AESA radar; ALQ-99 TJS pods |
| **Performance Envelope** | Based on F/A-18F Super Hornet: Ceiling: 50,000 ft; Max speed: Mach 1.8; Combat radius: 450+ nm; Crew: 2 (pilot + EWO) |
| **Datalinks / Comms** | Link-16; SATCOM; MIDS-LVT; ARC-210 multi-band radio |
| **PED Architecture** | NSA/CSS for SIGINT take from ALQ-218 passive collection; NAVAIR EW/Intel cells; VAQ squadron intelligence |
| **Tasking Authority** | JFACC/CFACC for escort/standoff jamming; CVW CAG |
| **Primary Mission Set** | Suppression of Enemy Air Defenses (SEAD); Airborne Electronic Attack (AEA — jamming); passive SIGINT collection during ingress/egress; force protection; escort jamming for strike packages |
| **Note for Collection Managers** | The ALQ-218 passive collection is a significant but secondary function; primary role is jamming. SIGINT product from ALQ-218 supports emitter geolocation and electronic order of battle compilation. |
| **Limitations** | High operational cost; carrier-dependent basing; primary mission is jamming, not SIGINT exploitation; ALQ-218 collection subordinated to tactical EW mission |
| **Public Sources** | [NAVAIR EA-18G](https://www.navair.navy.mil/product/EA-18G-Growler) | [Wikipedia EA-18G](https://en.wikipedia.org/wiki/Boeing_EA-18G_Growler) |

### 9.2 F-35A/B/C (Sensor Fusion / Multi-INT Contribution)

| Field | Details |
|---|---|
| **Designation** | F-35A (USAF); F-35B (USMC STOVL); F-35C (USN carrier) |
| **Operator** | USAF, USN, USMC (primary US); 17+ allied nations |
| **Type** | Multi-Role Stealth Fighter with Integrated Sensor Fusion / Multi-INT ISR Contribution |
| **Collection Capability** | **AN/AAQ-37 EODAS** (Electro-Optical Distributed Aperture System — 6× IR cameras in 360° spherical array, missile detection, day/night SA); **AN/AAQ-40 EOTS** (Electro-Optical Targeting System — FMV, FLIR, laser spot tracker/designator); **AN/APG-81 AESA** radar (SAR-like surface mapping, GMTI, inverse SAR); **AN/ASQ-239** (Barracuda) EW/SIGINT system — ESM, ELINT geolocation, electronic warfare; **CNI (Communications, Navigation, Identification)** integrated suite; sensor fusion engine merges all sensor data; multi-aircraft datalinks for collaborative target development |
| **Sensor Names** | AN/AAQ-37 EODAS DAS; AN/AAQ-40 EOTS; AN/APG-81 AESA; AN/ASQ-239 EW/SIGINT |
| **Performance Envelope** | A variant: supercruise ~Mach 1.2; max speed Mach 1.6; ceiling 50,000 ft; combat radius 590–760 nm; stealth (LO design) |
| **Datalinks / Comms** | MADL (Multifunction Advanced Data Link — stealthy F-35 to F-35 link); Link-16; SATCOM; ROVER; CDL-compatible via pod |
| **PED Architecture** | F-35-to-F-35 via MADL forms sensor network; feeds to AF DCGS via satellite downlinks; theater collection managers can task via ATO for ISR products |
| **Tasking Authority** | JFACC via ATO (primary); ISR collection "bonus" from standard strike sorties |
| **Primary Mission Set** | Air superiority; multi-role strike; but delivers persistent ISR as organic capability during combat operations; SIGINT/ELINT collection during ingress contributes to electronic order of battle |
| **Collection Manager Note** | F-35 is not primarily tasked as ISR; however, EODAS 360° IR provides passive MASINT/missile warning; ASQ-239 provides SIGINT; AESA provides GMTI-like capability; data shared through MADL mesh increases SA without dedicated ISR sorties |
| **Limitations** | Stealth constraints limit external sensor pods; primary role is combat; ISR collection subordinated to strike mission; MADL is F-35-only (limited interop with non-F-35 assets) |
| **Public Sources** | [Raytheon EODAS](https://www.rtx.com/raytheon/what-we-do/air/eodas) | [JSF.mil DAS](https://www.jsf.mil/das) | [Lockheed Martin DAS selection](https://news.lockheedmartin.com/2018-06-13-Lockheed-Martin-Selects-Raytheon-to-Deliver-Next-Generation-F-35-Sensor-System) |
