## 11. Ground & SOF Collection Systems

### 11.1 PROPHET / PROPHET Enhanced / Terrestrial Layer System (TLS)

| Field | Details |
|---|---|
| **Designation** | PROPHET Enhanced (current generation); TLS-BCT (Terrestrial Layer System — Brigade Combat Team) successor |
| **Operator** | USA (Brigade Combat Team level — MI Battalions) |
| **Type** | Ground-Based Tactical SIGINT / EW System |
| **Collection Capability** | **PROPHET Enhanced**: all-weather, near-real-time ground-based tactical SIGINT and EW; COMINT (communications interception) + ELINT; wideband BLOS capability; stationary, on-the-move, and manpack operations; modular for multiple vehicle platforms; collects full spectrum of enemy communications on battlefield |
| **TLS-BCT (Successor)**: | First integrated SIGINT/EW/cyber platform at BCT level; Lockheed Martin system; Stryker-mounted and manpack variants; SIGINT + EW + cyber operations in single system; first units fielded summer 2024 (TLS-BCT Manpack) |
| **Performance Envelope** | Vehicle-mounted (HMMWV/Stryker) and manpack; battalion/brigade deployment; extended-range BLOS via satellite data links |
| **Datalinks / Comms** | SATCOM BLOS; tactical data links to intelligence cells and DCGS-A |
| **PED Architecture** | DCGS-A; Army MIOC; NSA/CSS for national-level SIGINT products |
| **Tasking Authority** | BCT S2; MI Battalion commander; Brigade G2 |
| **Primary Mission Set** | Battlefield SIGINT for enemy comms intercept; emitter location/geolocation; targeting data for fires; force protection indications & warnings; EW jamming support |
| **Limitations** | LOS/BLOS constraints; platform electromagnetic signature; in contested environments, active collection increases detectability |
| **Public Sources** | [General Dynamics PROPHET](https://gdmissionsystems.com/intelligence-systems/signals-intelligence/prophet) | [PEO IEW&S TLS-BCT](https://cpeisw.army.mil/2024/12/11/278232/) | [Defense Scoop TLS](https://defensescoop.com/2024/03/21/army-funding-tls-bct-integrated-cyber-ew-sigint-system/) | [Lockheed Martin TLS-EAB](https://www.lockheedmartin.com/en-us/products/converged-cyber-signit-ew.html) |

### 11.2 TROJAN SPIRIT / TROJAN Network

| Field | Details |
|---|---|
| **Designation** | TROJAN SPIRIT II (legacy); TROJAN Network (current SES Space and Defense contract) |
| **Operator** | USA (INSCOM; MI Brigades; all echelons from corps to BCT) |
| **Type** | Intelligence SATCOM Terminal / Intelligence Network Enabler |
| **Collection Capability** | Not a sensor — a **communications/dissemination enabler**: secure commercial and military SATCOM providing national-to-tactical connectivity; links theater intelligence operations to NSA/DIA/NGA national agencies; extends JWICS and DoD secure networks to tactical units; enables split-based operations |
| **Performance Envelope** | Provides 14 channels digital voice/data (SCI and GENSER); max 1.544 Mbit/s aggregate data rate; SIPRNet/JWICS/NSA network access; satellite comms hub |
| **PED Architecture** | Acts as PED enabler connecting DCGS-A to national intelligence community; INSCOM operational network |
| **Tasking Authority** | INSCOM; MI Brigade G2; BCT S2 |
| **Primary Mission Set** | Connectivity for intelligence community — ensures tactical commanders receive national-level intelligence products; enables real-time intelligence dissemination to deployed forces |
| **Limitations** | Not a collector — a dissemination enabler; satellite bandwidth constraints in bandwidth-constrained environments |
| **Public Sources** | [FAS Trojan Spirit](https://irp.fas.org/agency/army/mipb/1996-3/steppark.htm) | [SES TROJAN contract](https://sessd.com/gsr/trojan-bringing-commercial-innovation-to-army-intelligence/) |

### 11.3 REMBASS-II (AN/GSR-8) — Unattended Ground Sensors

| Field | Details |
|---|---|
| **Designation** | AN/GSR-8 REMBASS-II (Remotely Monitored Battlefield Sensor System II) |
| **Operator** | USA (division, brigade, battalion levels); USMC; SOF |
| **Type** | Ground-Based Unattended Ground Sensor (UGS) System — MASINT |
| **Collection Capability** | Passive detection, classification, and direction-of-travel determination of personnel and vehicles; sensor suite: (1) seismic/acoustic sensor (personnel at 75m, vehicles longer range); (2) PIR (passive infrared) module — directional movement cuing; (3) magnetic sensor — ferrous target detection at 3m; LPI (Low Probability of Intercept) data transmission up to 15 km; can be unattended for up to 30 days |
| **Performance Envelope** | Self-contained; detects: personnel at 75m (seismic), vehicles at 300m; transmission range: 15 km LPI radio; battery life: 30 days; REMBASS-II vs legacy I-REMBASS: smaller and lighter |
| **Datalinks / Comms** | LPI radio transmitter (VHF band); receiver/display unit at monitoring station |
| **PED Architecture** | Brigade/battalion S2 monitoring; feeds PIR/targeting data to intelligence cells |
| **Tasking Authority** | BCT S2; MI Brigade; SOF team leader |
| **Primary Mission Set** | Area denial monitoring; route surveillance; border crossing detection; perimeter security; cueing of other ISR assets (UAVs, radars); named area of interest (NAI) monitoring |
| **Limitations** | Passive only — no active sensors; limited range; environmental noise can increase false alarm rate; requires physical emplacement (may require SOF infiltration) |
| **Public Sources** | [REMBASS-II Data Sheet](https://cdn.thomasnet.com/ccp/00713997/107924.pdf) | [FAS REMBASS](https://man.fas.org/dod-101/sys/land/rembass.htm) | [Defense-Update REMBASS-II](https://defense-update.com/20060107_rembass-ii-remotely-monitored-battlefield-sensor-system.html) |

### 11.4 SOF Collection Contributions (Doctrinal Summary)

| Element | Collection Role |
|---|---|
| **Civil Affairs (CA) Teams** | Civil information collection; assessment of civilian population; key leader engagement that generates HUMINT-relevant information; formal reporting via HUMINT channels |
| **Military Information Support Operations (MISO)** | Audience analysis generates population-level intelligence; MISO interaction with target audiences produces feedback on adversary influence and population sentiment |
| **SFOD-A / ODA teams** | HUMINT collection primary; tactical questioning; biometric collection; site exploitation; partner force debriefing |
| **Psychological Operations (PSYOP)** | Target audience analysis → OSINT; feedback collection from influence activities |
| **Ranger / SOF Direct Action** | Site exploitation (SSE — Sensitive Site Exploitation); biometric collection; document/material exploitation; supports TECHINT and DOCEX |
| **SOCOM ISR assets** | MQ-9A/B, RQ-20 Puma, Switchblade; coordinate through SOF JOC; tasked separately from conventional force ATO |

**Doctrinal Reference:** JP 3-05 (Special Operations); FM 3-05.30 (MISO); ATP 2-22.4 (Technical Intelligence)
