## Foundational Anchor

The intelligence disciplines are defined in **JP 2-0, Appendix B**. See the [JP 2-0 page](../02-joint-doctrine/jp-2-0-joint-intelligence.md) for the doctrinal definitions and Functional Manager assignments (ICD 113). Functional Manager responsibilities are formalized in [ICD 113](https://www.dni.gov/index.php/what-we-do/ic-policies-reports/intelligence-community-directives).
